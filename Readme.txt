Anon Ghost

A secure communication medium for those in liberty voids. Masks are generated automatically with the users details or with a  mask-seed to create private identities so users can create trusted relationships.

System requirements

* ImageMagick
* PHP
* MySQL

Check out http://anon.gho.st for a working version.
